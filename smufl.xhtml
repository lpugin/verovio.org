Music fonts

Verovio follows the Standard Music Font Layout (SMuFL) specification. It means that it is easily possible to change the music font used in Verovio and to have a personalised output. Verovio includes the Leipzig font, its own SMuFL-compliant music font.

It also supports and includes the Bravura font designed by Daniel Spreadbury. The Gootville font is a MuseScore font and is also included by default in Verovio.

Once included, a font can be selected by setting the font option accordingly. For example, the Bravura font can be selected with the --font=Bravura option in the command-line tool or by adding { font: "Bravura" } in the JavaScript toolkit options.
